# Security Policy

## Supported Versions

Mneno is pre-1.0. Security fixes are applied to the main development line.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities. Report suspected vulnerabilities privately to the project
maintainers.

Include:

- A description of the issue.
- Steps to reproduce.
- Impact and affected versions, if known.
- Any suggested mitigation.

The maintainers will acknowledge reports and coordinate fixes before public disclosure when appropriate.
