# Mneno Documentation

This directory contains the Mintlify documentation for the Mneno Core SDK.

## Local Preview

Run commands from this directory:

```bash
npx mint dev
```

Open `http://localhost:3000` to preview the site.

## Validation

Check internal links before publishing:

```bash
npx mint broken-links
```

Navigation and site configuration live in `docs.json`. Documentation changes pushed to the default branch are deployed
through the configured Mintlify GitHub integration.
